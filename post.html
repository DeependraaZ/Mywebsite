Gharbeti Application

This project was a college assignment project build in an android studio.
Gharbeti application reduces problem faced by both buyer and seller while buying and selling property. Most of the people are worried about the long process of searching property. Gharbeti Application is a meeting point where buyer and seller of property from different places find each other.
Gharbeti Application deals with Property buy/sell/rent which helps to find buyer and seller of the property.

Technologies:
  • - Java
  • - Node
  • - Express
  • - MongoDB
